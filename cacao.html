Cacao Web Design

User experience and interface

My work on this project focused on the research and planning stages for the creation of the website. I used tools such as user personas, target audience definition, empathy maps, and an analysis of the current competitive market.
Starting from the site map, I gained clarity on how the low-fidelity wireframes should be structured, allowing me to move forward to the final prototype phase. This project helped me refine my skills in prototyping (Figma) and usability testing prior to code development.
